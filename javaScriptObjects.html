JavaScript Objects

Primitive data types as objects

Objects have methods and other properties. You can access them using the object operator A period "."

Boolean object

  • var boolFlag1 = new Boolean();
  • var boolFlag2 = new Boolean(0);
  • var boolFlag3 = new Boolean(1);
  • var boolFlag4 = new Boolean(false);
  • var boolFlag5 = new Boolean(true);
  • var boolFlag6 = new Boolean("");
  • var boolFlag7 = new Boolean("false");

All objects have "valueOf()" and "toString()"

Number object

Static properties used for testing overflow conditions

  • Number.MAX_VALUE
  • Number.MIN_VALUE
  • Number.NaN (Not a Number)
  • Number.NEGATIVE_INFINITY
  • Number.POSITIVE_INFINITY

Instance methods

  • toExponential()
  • toFixed()
  • toPrecision()
  • toLocaleString()
